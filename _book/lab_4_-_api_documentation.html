Lab 4 - API Documentation

In this lab, you will generate and test an ActiveDocs documentation for the API created in the 2nd lab.
  1. Open a web browser and go to https://3scale-admin.3scale[your instance #].rhtechofficelatam.com/
  2. Login as admin/admin
  3. Click on the APIs tab.
  4. Click on the AciveDocs link.

  1. Click on the Create a new spec.

  1. Enter the following values:

  2. Name: Products

  3. System Name: products_spec
  4. Publish?: checked
  5. Description: Products API Documentation

  1. Open a new web browser tab, and go to http://editor.swagger.io/
  2. Select File -> Import URL.
  3. Enter the following URL: http://products.3scale[your instance #].rhtechofficelatam.com/rest/swagger.yaml
  4. Click OK.
