Lab 6 - Developer Portal

In this lab you will work on the Developer Portal for the API’s customers/partners/users.
  1. Open a web browser.

  2. Download this two files:

  3. https://github.com/pszuster/3ScaleTD/raw/master/DevPortal/RHMartLogo.png

  4. https://github.com/pszuster/3ScaleTD/raw/master/DevPortal/RHMartBackground.jpg

  5. Go to https://3scale-admin.3scale[your instance #].rhtechofficelatam.com

  6. Login as admin/admin
  7. Click on the Developer Portal tab.
  8. Click on the arrow next to New Page button.
  9. Click on New File.

  1. Set Section to images.
  2. Set Path to /images/RHMartLogo.png
  3. Click on the Choose File button.
  4. Select the RHMartLogo.png file you downloaded in the first step.
  5. Click on the Create File button.

  2. Set Path to /images/RHMartBackground.jpg

  5. Click on the Create File button.
